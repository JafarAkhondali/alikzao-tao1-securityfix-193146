Introduction
============
This asynchronous framework with a modular structure like Django. But with mongodb, jinja2, websockets, and more than a simple barrier to entry.

Framework Installation
----------------------

::

   $ pip install tao1


Getting Started
---------------

Create project::

   $ utils.py -p name

Create apps::

   $ utils.py -a name
